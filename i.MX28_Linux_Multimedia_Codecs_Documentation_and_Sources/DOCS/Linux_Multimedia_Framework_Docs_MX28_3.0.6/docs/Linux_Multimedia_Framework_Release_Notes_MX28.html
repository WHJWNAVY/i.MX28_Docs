Known Issues

  • Software video decoders have the performance limitation: in common, on i.MX28 EVK platform up to QVGA (320x240) size video can be run smoothly.
  • AAC decoder: ADIF format does not support seek mode.
  • The WMA decoder have the performance limitation: Only 2 channels and bit rates up to 196Kbps can be run smoothly
  • Some audio clips with multi-channel may have some performance limitation.
  • For some critical video cases, there might be some performance issue in particular point.

Gstreamer-based Plugins List

 

You will find following gstreamer-based plugins after build.

 

  • aiurdemux: aiurdemux: Aiur universal demux
  • beepdec: beepdec: unify audio decoder
  • mfw_mp3decoder: mfw_mp3decoder: freescale mp3 decoder
  • mfw_aacdecoder: mfw_aacdecoder: Freescale AAC Decoder Plugin
  • mfw_vorbis_decoder: mfw_voirbisdecoder: Freescale vorbis Decoder Plugin
  • mfw_h264decoder: mfw_h264decoder: Freescale H264 decoder
  • mfw_mpeg4aspdecoder: mfw_mpeg4aspdecoder: Freesclae MPEG plugin
  • mfw_mp3encoder: mfw_mp3encoder freescale mp3 encoder

 

license limited:

  • mfw_wma10decoder: mfw_wma10decoder: Freescale's wma10 decoder
  • mfw_aacplusdecoder. mfw_aacplusdecoder: Freesales AAC plus decoder
